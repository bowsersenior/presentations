This is a ShowOff presentation (http://github.com/bowsersenior/intro_to_redmine_presentation)

If you want to see it, you can either:

visit http://gentle-sky-70.heroku.com

OR

install showoff and then run it in this directory:

  $ gem install showoff
  $ git clone git@github.com:bowsersenior/intro_to_redmine_presentation.git
  $ cd intro_to_redmine_presentation
  $ showoff serve

Then you should be able to go to http://localhost:9090 to view the
presentation.
